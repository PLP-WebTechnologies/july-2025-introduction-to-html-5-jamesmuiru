This page is an assignment demonstrating semantic structure, accessibility, and beginner SEO principles.

  • Accessibility

Semantic Structure

The page uses HTML5 semantic elements such as <header>, <nav>, <main>, <section>, and <footer>. These provide meaningful structure that benefits both users and search engines.

Accessibility

Accessibility is improved through the use of proper landmarks, logical heading order, and attributes like aria-label to help screen readers navigate the page effectively.

Search Engine Optimization (SEO)

The page includes a descriptive <title> and <meta description>. Headings are structured in a logical hierarchy, and the content is clear, readable, and crawler-friendly.
